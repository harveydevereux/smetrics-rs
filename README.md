### API tokens

- Bluesky has a pubic API (AT protocol) so there is no need if you just want to read posts as here.

- Tumblr is very [light touch](https://www.tumblr.com/oauth/apps) for read-only actions.

- Instagram requires more.
  - First you must have a "professional account". You can make your personal account a creator account.
  - Then register as a Meta developer [via a Facebook account](https://developers.facebook.com).
  - Then create a [new app](https://developers.facebook.com/apps/)
  - Give it the "Manage messaging and Instagram content" use case permissions.
  - Add your instagram as a tester.
  - Finally, generate an access token (which will expire etc???).
  - It does not need to be published.
